# Security Audit: Botium Toys

**Internal Security Assessment**

Completed as part of the Google Cybersecurity Professional Certificate.

---

### Executive Summary
Conducted a comprehensive security audit for Botium Toys to evaluate their current security controls and compliance posture. Identified multiple security gaps and provided prioritized recommendations to reduce risk.

### Audit Objectives
- Assess current security controls.
- Identify vulnerabilities and risks.
- Evaluate compliance requirements.
- Provide actionable remediation steps.

### Key Findings
- All employees have access to customer data.
- Lack of proper access controls and role-based permissions.
- Insufficient network segmentation.
- Password policies need to be strengthend.
- Logging and monitoring need improvement.

### Recommendations
- Implement least privilege access controls.
- Deploy role-based access control.
- Enforce strong password requirements and Multi Factor Authentication.
- Improve logging and monitoring capabilities.
- Update security policies and documentation.

### Skills Demonstrated
- Security auditing and risk assessment.
- Technical documentation.
- Understanding of access control principles.
- Compliance and regulatory knowledge.
